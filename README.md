📝 Overview
This project is a **Mall Management System** developed as part of the Fortgeschrittene Programmierungs course at Babeș-Bolyai University. 

✨ Key Features
* **Store Management:** Add, update, and remove stores across different mall floors/categories.
* **Staff Directory:** Manage employee records, roles, and contact information.
* **Search Functionality:** Quickly find stores or employees using specific filters.

🛠️ Tech Stack
* **Language:** Java 
* **Database:** MySQL
